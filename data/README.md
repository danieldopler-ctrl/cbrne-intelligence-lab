# Data Handling

`raw/` stores local source snapshots during development and is ignored by git. `processed/` stores derived local exports and is also ignored.

Only small, explicitly safe test fixtures may be committed under `test-fixtures/`. Real public records still require provenance, permitted-use review, and source limitation documentation before any public sharing.

`test-fixtures/chem-hazmat-safe-fixture.csv` contains fictional records solely for interface and rule testing. It is not evidence and is not an operational dataset.
